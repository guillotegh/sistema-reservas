# Configuración del Sistema de Reservas

Este proyecto ha sido migrado a una arquitectura moderna con Vite, React y Supabase.

## 1. Instalación de Dependencias

Abre una terminal en esta carpeta y ejecuta:

```bash
npm install
```

## 2. Configuración de Supabase

1. Crea una cuenta gratuita en [supabase.com](https://supabase.com).
2. Crea un nuevo proyecto.
3. Ve a **SQL Editor** y ejecuta el siguiente script para crear la tabla de reservas:

```sql
create table reservas (
  id bigint generated by default as identity primary key,
  "fechaCreacion" date default current_date,
  "fechaViaje" date,
  "fechaRegreso" date,
  titular text,
  destino text,
  operador text,
  "precioVenta" numeric,
  "precioNeto" numeric,
  moneda text default 'ARS',
  "liquidacionRecibida" boolean default false,
  "voucherEnviado" boolean default false,
  "pagosCliente" jsonb default '[]'::jsonb,
  "pagosProveedor" jsonb default '[]'::jsonb
);
```

4. Ve a **Project Settings > API**.
5. Copia la `URL` del proyecto y la clave `anon` / `public`.

## 3. Variables de Entorno

Crea un archivo llamado `.env.local` en la raíz del proyecto y pega tus credenciales:

```env
VITE_SUPABASE_URL=tu_url_de_supabase
VITE_SUPABASE_ANON_KEY=tu_clave_anonima
```

## 4. Ejecución

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

Abre tu navegador en la URL que aparece en la terminal (usualmente `http://localhost:5173`).

## 5. Correo y Contraseña
Por defecto, Supabase requiere confirmar el email. Puedes desactivar "Confirm email" en **Authentication > Providers > Email** para pruebas rápidas, o crear un usuario desde el panel de Supabase.
